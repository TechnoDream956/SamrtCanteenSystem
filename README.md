# 🍽️ B.U Eats — Smart Canteen System

A full-stack food ordering platform built for Bennett University campus, featuring real-time order tracking, OTP authentication, and a staff management dashboard.

🔗 **Live Demo**: https://samrt-canteen-system.vercel.app

## Features

- 🛒 Browse 4 campus canteens and place orders instantly
- 📡 Real-time order tracking via WebSockets
- 🔒 OTP-based authentication for secure student login
- 👨‍🍳 Staff dashboard for managing orders and daily reports
- 📱 Fully responsive mobile-friendly design

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | HTML · CSS · JavaScript |
| Backend | Python · Flask |
| Real-time | WebSockets |
| Auth | OTP-based JWT |
| Hosting | Vercel |
